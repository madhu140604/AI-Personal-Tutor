Basic Python Course

1. Introduction to Python

What is Python?
Python is a high-level, interpreted programming language known for its readability and wide applicability in fields such as web development, data science, AI, and more.

History of Python:
Developed by Guido van Rossum and first released in 1991, Python was designed to emphasize code readability and simplicity.

Features of Python:
Easy to learn and use, dynamically typed, extensive libraries, portable, open source, and supports multiple programming paradigms.

Applications of Python:
Used in web development, automation, data analysis, machine learning, scripting, and desktop applications.

Installing Python:
Download from the official website (python.org), follow the installation instructions, and verify using python --version in the terminal.

Writing your first Python program:
Open a text editor or IDE, write print("Hello, World!"), save the file with a .py extension, and run it using the Python interpreter.

2. Python Basics

Python Syntax:
Python uses indentation to define blocks of code. No need for curly braces or semicolons.

Comments in Python:
Single-line comments start with #, multi-line comments use triple quotes ('''comment''').

Variables and Data Types:
Variables are created when first assigned. Common types include int, float, str, bool, list, tuple, dict, and set.

Type Casting:
Convert between types using int(), str(), float(), etc.

Basic Input and Output:
Use input() to read input and print() to display output.

String Operations:
Concatenate using +, repeat with *, access characters using indexing, and manipulate with string methods like lower(), upper(), strip(), etc.

Basic Operators:
Includes arithmetic (+ - * / % // **), comparison (== != > < >= <=), and logical (and or not) operators.

3. Control Flow

Conditional Statements:
Use if, if-else, and elif to execute code based on conditions.

if x > 0:
  print("Positive")
elif x == 0:
  print("Zero")
else:
  print("Negative")

Loops:
Use for and while loops to iterate over sequences or execute blocks while conditions are true.

for i in range(5):
  print(i)

Loop Control Statements:
break exits the loop, continue skips the current iteration, and pass does nothing (placeholder).

Nested Loops and Conditionals:
Use indentation carefully to nest logic for complex flows.

Using range():
The range() function generates sequences of numbers for iteration.
