7. File Handling

Working with Files:
Python allows you to read, write, and manipulate files using built-in functions like open().

Opening a File:

f = open("example.txt", "r")
print(f.read())

Modes: r - read, w - write, a - append, x - create.

Reading Files: Use read(), readline(), or readlines() methods.

Writing to Files: Use write() or writelines().

f = open("example.txt", "w")
f.write("Hello Python")
f.close()

Using with statement: Automatically closes the file after operation.

with open("example.txt") as f:
    data = f.read()
    print(data)

8. Exception Handling

What is Exception Handling?
It is the process of handling runtime errors so the program doesnâ€™t crash unexpectedly.

Using try-except:

try:
    x = 10 / 0
except ZeroDivisionError:
    print("You can't divide by zero!")

Multiple Exceptions:

try:
    x = int("abc")
except ValueError:
    print("Invalid conversion!")

Finally Block: Executes code regardless of exception.

try:
    print("Trying")
finally:
    print("Finally block executed")

Raise Exception: Use raise to throw an exception.

raise ValueError("This is a custom error")
